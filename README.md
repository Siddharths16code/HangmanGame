# 🎯 CodeAlpha Hangman Game (Task 1)

A simple text-based Hangman game built in Python as part of the **CodeAlpha Internship Task 1**. The player guesses a randomly selected word one letter at a time using the command line interface (CLI).

---

## 🧩 Features

- Randomly selects one word from a predefined list of 5 words.
- Allows up to 6 incorrect guesses.
- Displays progress after each guess (correct letters + underscores).
- No use of files, APIs, or external libraries.

---

## 📌 Task Scope (As per Instructions)

- ✅ Used a small list of 5 predefined words.
- ✅ Limited incorrect guesses to 6.
- ✅ Built using **basic console input/output (CLI)** — no GUI or audio used.

---

## 🛠️ Key Concepts Used

- `random`
- `while` loop
- `if-else` statements
- `strings` and `lists`
